Requirements

  1. Java 17
  2. Maven installed
  3. MySql server

Running the application locally

git clone https://github.com/jair921/quasar.git
cd quasar
# On src/main/resources/application.properties configure database connection
mvn spring-boot:run

URL Service

https://fire-quasar-jair.herokuapp.com/

Rest endpoints

    "satellites": [

            "name": "kenobi",
            "distance": 100.0,

Top secret split (store)

Request

POST /topsecret_split/{satellite_name}
{
    "distance": 142.7,
    "message": [
                "este",
                "",
                "un",
                "",
                ""
            ]
}

Response

{
    "position": null,
    "message": "Store split message"
}

Top secret split

Request

GET /topsecret_split
{
    "distance": 142.7,
    "message": [
                "este",
                "",
                "un",
                "",
                ""
            ]
}

Response

{
    "position": [
        59.634420899438396,
        -30.932718106937056
    ],
    "message": "este es un mensaje secreto"
}

Remove old split messages

Request

POST /restart_split

Response

{
    "position": null,
    "message": "Remove all split messages."
}
